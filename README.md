# Secure Production Linux Server on AWS

## Project Overview
This project demonstrates how to deploy and secure a production-ready Linux server on AWS using EC2. The focus is on security, reliability, and real-world best practices.

## Technologies Used
- AWS EC2 (Free Tier)
- Ubuntu Linux
- Nginx
- UFW Firewall
- SSH Hardening
- Let's Encrypt SSL

## Project Objectives
- Launch and configure an EC2 instance
- Secure SSH access
- Configure firewall rules
- Deploy and secure a web server
- Enable SSL for secure communication
- Monitor logs and system health
